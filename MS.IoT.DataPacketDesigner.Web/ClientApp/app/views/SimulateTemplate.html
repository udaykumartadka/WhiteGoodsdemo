Simulating Your Product

We will simulate your IoT Experience in the cloud. Just follow these steps.
Step 1
Order 4x4 Device
If you already have the 4x4 device start with step 2, otherwise click the below link to order a new one.
Step 2
Setup 4x4 Device
Please download the tools required to get started with 4x4. Please follow the get started document to configure and setup 4x4 device on your machine.
Step 3
Connect UART Cable to 4x4
Connect the UARt cable to 4x4 device. Follow the instructions on this link
Step 4
Download and Deploy the 4x4 code on device
Download the C code below to deploy to 4x4 device. You will need Visual Studio 2017 V 15.3 or later to open the solution and deploy the code.
Step 5
Download and Install the Simulator Desktop App
The simulator app will help you connect to the 4x4 device and send data packets to the IoT hub.
